# Technology Research Sample

This folder contains an example of technology research conducted using the `AGENTS_RESEARCH.md` template.

## Contents

- **AIRTABLE_CUSTOM_EXTENSIONS.md** - Complete technology evaluation for Airtable Custom Extensions

## About This Example

This research was conducted to evaluate Airtable Custom Extensions for a project that:
- Provides interfaces for engineers to submit weekly updates
- Aggregates and presents reports to stakeholders
- Prioritizes usability, scalability, and maintainability

## Key Sections

The research document includes:

1. **Technical Evaluation** - Comprehensive analysis of:
   - Platform requirements and availability
   - Pricing and licensing implications
   - Technical implementation details
   - Code samples and architecture patterns
   - Risks and limitations

2. **Business Analysis** - Decision-maker focused content:
   - ROI calculations
   - Cost comparisons
   - Stakeholder talking points
   - Implementation timeline
   - Success criteria

3. **Role-Based Perspectives**:
   - **Technical Analysis** (GPT-5.2 role) - Deep-dive into feasibility, architecture, and implementation
   - **Business Review** (Opus 4.5 role) - Simplified findings with business relevance

## Using This Template

To conduct your own technology research:

1. Copy the structure from `../../AGENTS_RESEARCH.md`
2. Fill in each section with technology-specific details
3. Include both technical and business perspectives
4. Provide actionable recommendations
