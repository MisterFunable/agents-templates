# Poetry Examples

Demonstrations of AGENTS_POETRY.md guidelines in practice.

## Examples Included

| File | Form | Demonstrates |
|------|------|--------------|
| `morning_sonnet.txt` | Shakespearean Sonnet | Traditional form, iambic pentameter, concrete imagery |
| `autumn_haiku.txt` | Haiku | 5-7-5 structure, nature focus, seasonal reference |
| `ordinary_things.txt` | Free Verse | Enjambment, white space, conversational tone |
| `revision_example.txt` | Before/After | Revision process, removing clichés, strengthening imagery |

## Key Techniques Shown

**Concrete Imagery:**
- "coffee cup" and "morning light" instead of "happiness"
- "Gray clouds pressed low" instead of "sad day"

**Sound Devices:**
- Alliteration: "Whispering winds weave through willows"
- Assonance: "rain remains, gray refrain"

**Line Breaks:**
- Breaking for emphasis ("goodbye")
- Enjambment for flow and tension

**Revision:**
- Removing filler words ("really", "very")
- Replacing abstractions with sensory details
- Eliminating clichés with fresh comparisons

## Common Mistakes Avoided

| Mistake | How Examples Avoid It |
|---------|----------------------|
| Clichés | Fresh metaphors ("heart like bread" not "heart of gold") |
| Telling emotions | Showing through details and actions |
| Forced rhyme | Meaning guides form, not vice versa |
| Abstract language | Concrete sensory details throughout |
| Ignoring line breaks | Intentional breaks for emphasis and breath |

## Usage

Read these examples alongside AGENTS_POETRY.md to see guidelines in action.

Each example demonstrates multiple techniques, showing how they work together in complete poems.
