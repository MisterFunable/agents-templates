# Data Processor

Simple Python example demonstrating AGENTS_PYTHON.md guidelines.

Convert CSV files to JSON with validation and logging.

## Setup

**Requirements:** Python 3.9+

```bash
cd examples/python-sample
python data_processor.py input.csv output.json
```

## Usage

```bash
# Basic conversion
python data_processor.py data.csv

# Save to file
python data_processor.py data.csv output.json
```

## Features Demonstrated

This example shows:
- Proper type hints
- Path handling with pathlib
- Structured logging
- Error handling with specific exceptions
- Module-level docstrings
- CLI argument parsing
